# Notes – Android Notes App

A multi-screen Android notes app for creating, viewing and deleting notes.

## Features
- Create notes with a title and content
- View all saved notes in a scrollable list
- Delete notes via long-press confirmation dialog or spinner selection
- Data persistence using SharedPreferences
- Multi-activity architecture (MainActivity, AddNoteActivity, DeleteNoteActivity)

## Tech Stack
- Java
- Android SDK
- SharedPreferences (local storage)
- Gradle (Kotlin DSL)

## How to run
1. Clone the repo
2. Open in Android Studio
3. Run on emulator or physical device
